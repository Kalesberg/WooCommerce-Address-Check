AC Proof is woocommerce integrated plugin which scrutinizes addresses during checkout

== Description ==

Addres Check Proof(AC Proof) is a very powerful assistant for verifying addresses over planet.
It removes any burdens of shop owners when delivering products to the customers by ensuring the billing/shipping addresses are correct
This plugin is just integrated with WooCommerce plugin which is most famous Wordpress E-Commerce plugin

== Features ==

Use Google Geocode API
Does not interrupt any plugins , no clashes with other plugins in wordpress
Supports proxy support
Provides very fast and exact verification of addresses
Intelligent address checking engine
Flexible database table management
Very easy to extend the plugin by your own codes
Clean administration settings

== Manual installation ==

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation¡¯s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

== Usage ==

1. Login to the wp-admin
2. After activating the plugin, go to that plugin settings page(Under Wordpress Settings Menu)
3. No mandatory options
   Set the plugin options as you wish and grab all the addresses in your hand
That's all! You are ready to go!
